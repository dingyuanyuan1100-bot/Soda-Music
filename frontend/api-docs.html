API DOCUMENTATION

汽水音乐第三方接口文档

当前网站前端部署在 80 端口，实际 API 服务运行在 http://47.116.42.11:1234。

返回主页 Markdown 文档 Swagger 接口清单 JSON

GET/api/health

健康检查，确认 API 和上游是否可用。

GET/api/routes

返回当前服务已暴露的接口列表。

GET/api/search

统一搜索接口。

参数：q=关键词&type=songs|playlists

GET/api/search/songs

搜索歌曲。

参数：q=关键词

GET/api/search/playlists

搜索歌单。首屏部分结果会直接补真实封面。

参数：q=关键词

GET/api/playlist/share-meta

按歌单 ID 获取分享页真实封面。

参数：id=歌单ID

GET/api/playlist/detail

获取歌单详情和歌曲列表。

参数：id=歌单ID

GET/api/song/detail

获取分享页歌曲详情。

参数：id=歌曲ID

GET/api/song/full

单首歌聚合接口，一次拿详情、歌词、可播性。

参数：id=歌曲ID&name=歌名&artist=歌手&album=专辑&cover=封面URL

GET/api/lyrics

获取歌词，优先上游，失败回退分享页。

参数：id=歌曲ID&name=歌名&artist=歌手&album=专辑

GET/api/playback/check

检测歌曲可播性与推荐播放源。

参数：id=歌曲ID&name=歌名&artist=歌手&album=专辑&cover=封面URL

GET/api/stream

音频流代理，可直接给 audio 使用。

参数：id=歌曲ID&name=歌名&artist=歌手&album=专辑&cover=封面URL
可选：preferred=share&shareUrl=...

GET/api/cover

图片代理，图片失效时返回占位图。

参数：url=图片地址

GET/api/auth/status / probe / pages / my-playlists-probe

本机登录态调试接口，用于研究客户端登录态复用，不作为正式稳定账号接口承诺。
